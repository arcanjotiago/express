# Description
This repository was created using `Express` framework.  Here we have three basic endpoints to perform some actions that illustrate how Express works.

# Endpoints
-`GET`  /status   
-`POST` /calcnumber  
-`POST` /checkmail

## Requisition examples
`POST` /calcnumber 
```
{
    "number": 100
}
```

`POST` /checkmail 
```
{
    "email": "examplE@gmail.com"
}
```
